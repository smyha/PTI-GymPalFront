
  # Gympal Frontend Design

  This is a code bundle for Gympal Frontend Design. The original project is available at https://www.figma.com/design/D94Oc3e2MdbzGxNIcjBvNe/Gympal-Frontend-Design.

## Running the code

1. Install dependencies with PNPM: `pnpm i`
2. Set up `VITE_API_BASE_URL` (e.g. `http://localhost:3000`)
3. Dev server: `pnpm dev`

### Optional: Generate OpenAPI Types

Run `pnpm generate:types` to generate `src/api/types.ts` from the backend OpenAPI.
